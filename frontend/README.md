# How to start frontend

## 1. Install Nodejs

See [Installation Guide](https://nodejs.org/en/download/)

## 2. Install required packages

```
npm install
```

### 3. Compiles and hot-reloads for development

```
npm run serve
```

-----

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

```
npm run test:unit
```

### Lints and fixes files

```
npm run lint
```
